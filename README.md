## GitHub Pages Setup

1. Open the `CODSOFT` repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. In `Build and deployment`, choose:
   ```text
   Deploy from a branch
   ```
5. Select:
   ```text
   main
   ```
6. Select folder:
   ```text
   /root
   ```
7. Click `Save`.

GitHub will create a website link for the project.
