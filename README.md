# CODSOFT

This repository contains CODSOFT internship tasks.

## Run Task 1 Chatbot

### Online

After enabling GitHub Pages, open:

```text
https://ruchitawagh-06.github.io/CODSOFT/
```

The page will automatically open the chatbot.

### Offline

Download or clone this repository, then open:

```text
Task-1-Chatbot/index.html
```

No installation is required for the website version.

## Run Task 2 Tic-Tac-Toe AI

Open:

```text
Task-2-Tic-Tac-Toe/index.html
```

## Run Task 3 Recommendation System

Open:

```text
Task-3/index.html
```

## GitHub Pages Setup

1. Open the `CODSOFT` repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. In `Build and deployment`, choose:
   ```text
   Deploy from a branch
   ```
5. Select:
   ```text
   main
   ```
6. Select folder:
   ```text
   /root
   ```
7. Click `Save`.

GitHub will create a website link for the project.
