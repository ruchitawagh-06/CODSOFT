# Task 2: Tic-Tac-Toe AI

A professional Tic-Tac-Toe web game where a human player plays against an AI agent.

## Features

- Human vs AI gameplay
- Unbeatable AI using the Minimax algorithm
- Optional Alpha-Beta pruning
- Difficulty modes
- Player can choose X or O
- Score tracking
- Clean professional interface
- Runs directly in the browser

## Run

Open this file in a browser:

```text
index.html
```

No installation is required.

## Algorithm

The AI checks all possible future moves using Minimax. It scores winning, losing, and draw outcomes, then chooses the best move.
