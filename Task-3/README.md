# Task 3: Global Movie Recommendation System

A browser-based recommendation system that suggests global movies based on user preferences.

## Features

- Content-based filtering recommendation ranking
- Genre, language, mood, era, and rating controls
- Search by title, country, language, or tag
- Quick filters for balanced, family, classic, new era, award, and regional picks
- Match percentage and recommendation explanation
- Cinematic movie-themed responsive website layout
- Sample catalog across Hindi, English, Korean, Gujarati, Tamil, Telugu, Japanese, Spanish, French, Malayalam, Bengali, Marathi, and more

## Run

Open this file in a browser:

```text
index.html
```

No installation is required.

## Method

The system scores each movie by comparing user preferences with movie features such as genre, language, mood, rating, year, country, and tags. The highest scoring movies are shown first.
